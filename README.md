# 高考志愿填报助手 - 微信小程序

## 页面展示

### 首页

登录遮罩 → 主页内容，包含 Hero 区域（品牌标语 + 快捷入口）、功能卡片（查院校/查专业/AI 助手）、数据统计展示。

| 登录页 | 主页 |
|-------|------|
| ![登录页](docs/images/login.png) | ![主页](docs/images/home.png) |
| 微信手机号一键登录 | 选对方向·成就未来 标语，开始查询 / AI 咨询按钮 |
| | 查院校、查专业、AI 助手三张功能卡 |
| | 2800+ 院校 / 700+ 专业 / AI 智能推荐统计 |

### 院校列表

![院校查询](docs/images/schools.png)

- 顶部搜索栏：按院校名称关键词搜索
- 筛选栏：按省份、院校标签（985/211/双一流）筛选
- 院校卡片：校名、标签（985/211 高亮）、所在地、类别
- 收藏按钮：点击可收藏/取消收藏院校
- 分页加载：已加载数 / 总数

### 专业列表

- 顶部搜索栏：按专业名称关键词搜索
- 筛选栏：按学科门类筛选
- 专业卡片：专业名称、代码、学科门类、学位、学制、描述
- 查看开设院校入口

### 院校详情

- 院校信息卡：名称、标签、代码、类别、省份、城市、简介
- 开设专业列表：专业名称、代码、类别、学位、学制、描述
- 智能推荐区：点击「开始推荐」获取冲刺/稳妥/保底建议，展示推荐院校、录取概率、推荐专业

### AI 助手

- 成绩表单：可输入总分、全省排名、省份
- 对话界面：AI 志愿填报咨询，支持多轮对话
- 消息气泡：区分用户 / AI 身份
- 输入框：文本输入 + 发送按钮

### 我的

![个人页](docs/images/profile.png)

- 顶部用户卡片：头像、昵称、手机绑定状态（登录后）
- 菜单列表：我的收藏、AI 助手、查院校、查专业
- 退出登录按钮（登录后显示）
- 版本号

### 收藏

- 收藏院校列表，支持取消收藏
- 点击进入院校详情
- 分页加载

## 技术栈

- 微信小程序原生框架
- JavaScript / WXML / WXSS
- 自定义 tabBar 导航
- 深色主题 UI 设计

## 启动

用微信开发者工具打开项目根目录，导入 `miniprogram` 目录即可预览。
